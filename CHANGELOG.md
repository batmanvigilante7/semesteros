# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-11

### Added
- Initial release of **SemesterOS** — Your Academic Operating System.
- Dashboard with next study recommendation engine and smart notification system.
- Course manager (formerly Subjects) with syllabus tracker and credit hours.
- Planner (formerly Tasks) with priority tags, status updates, and progress stats.
- Timeline (formerly Calendar) with interactive agenda view of deadlines.
- Insights page (formerly Analytics) displaying course completion and study duration streaks.
- Preferences dashboard (formerly Settings) for personalizing workspace term boundaries.
- UI Showcase for viewing custom design system components built on Tailwind v4 and Framer Motion.
